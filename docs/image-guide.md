# Eastokyo Image Guide

## File organization

Store each published story's images under `assets/images/stories/<story-slug>/`.

## Naming

Use descriptive lowercase filenames separated by hyphens. Reserve `hero` for the primary story image.

## Delivery

- Prefer WebP or AVIF for photographic images.
- Keep original files outside the public repository.
- Export only the dimensions and quality required by the site.
- Write useful alt text and retain photographer credits.
