# Eastokyo Publishing Guide

1. Finalize the article and photography locally.
2. Choose the permanent title and URL slug.
3. Create `stories/<story-slug>/index.html`.
4. Add optimized images to `assets/images/stories/<story-slug>/`.
5. Add the story metadata to `data/stories.js`.
6. Update `sitemap.xml`.
7. Test keyboard navigation, mobile layouts, links, captions, credits, and metadata.
8. Publish.
